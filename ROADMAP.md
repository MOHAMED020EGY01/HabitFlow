# Roadmap

## Near Term

- Stabilize the theme and appearance experience so the app behaves consistently across settings and runtime.
- Expand automated testing around reminders, overlays, widgets, and rollover behavior.
- Review permission UX and ensure the app explains why each capability is needed.

## Medium Term

- Add CI/CD automation for build verification and release packaging.
- Improve backup and data export/import support for user safety and portability.
- Refine performance profiling around startup, reminders, and widget updates.

## Longer Term

- Consider introducing a remote sync or cloud backup option if the app evolves beyond a local-first product.
- Expand analytics and crash monitoring in a privacy-conscious way.
- Continue improving accessibility and localization quality.
