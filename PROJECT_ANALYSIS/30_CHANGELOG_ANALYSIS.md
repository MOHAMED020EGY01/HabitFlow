| **v1** | تأسيس الجداول الأولية لـ `habits` و `habit_logs`. | إطلاق الهيكل الأساسي لتخزين العادات وسجل التتبع البسيط. |

| **v4** | إضافة فهرس بحث مخصص لحقل العادات النشطة وتاريخ الإنشاء. | تحسين أداء استعلامات لوحة التحكم الرئيسية Home. |
| **v5** | تعديل صيغة تخزين أوقات التنبيهات وإزالة الأقواس JSON. | تبسيط منطق التحويل وتفادي تعقيدات التحليل اللغوي للنصوص. |

| **v7** | إضافة فهرس بحث مخصص لحقل `startedAt` في جدول العادات. | تسريع فرز وترتيب العادات بحسب تاريخ البدء الفعلي. |
| **v8** | تأسيس وجدولة جدول الإشعارات وتاريخ التذكيرات `notifications`. | توفير واجهة لسجل التنبيهات الفائتة وتفادي تداخل الإشعارات. |
| **v9** | إضافة حقل آخر وقت للتوقف المؤقت للعادة `inactiveSinceTimestamp`. | حساب فترات ومدة توقف العادة ومزامنتها مع قطع Glance. |
| **v10** | إضافة حقل صوت المنبه المختار للعادة `reminderVoice`. | دعم اختيار نغمات تنبيه فريدة لكل عادة. |
| **v11 & v12** | تحديث وتحديث هاش الهوية لملف قاعدة البيانات Room. | ضمان سلامة الاتصال ونقل الجداول بأمان بين الهواتف. |

## تطور الميزات والمنصة البرمجية / Platform & Feature Evolution

* **الإطلاق الأول (Phase 1)**: تأسيس واجهات Jetpack Compose ونمط Clean Architecture.
* **إطلاق التنبيهات الصوتية والنوافذ العائمة (Phase 2)**: إدراج `HabitOverlayService` وتوفير نطق النصوص المخصص (Text-To-Speech).
* **إطلاق ودعم قطع الشاشة الرئيسية (Phase 3)**: إدراج Glance Widgets وتأسيس عمال المزامنة والتحديث المباشر للرسوم (Direct view composing).
* **إصلاحات دعم أندرويد 14+ (Phase 4)**: إضافة أنواع الخدمات الأمامية المحددة `specialUse` لتفادي حظر تشغيل الخدمات الخلفية في نظام تشغيل أندرويد 14 وما فوق.
